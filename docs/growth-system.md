# 루미의 성장 시스템

루미의 성장은 모델 파인튜닝이 아니라 운영 구조의 누적으로 이루어집니다.

## 1. 페르소나

AI 비서의 이름, 말투, 호칭, 작업 원칙을 정합니다. 페르소나는 매번 같은 비서와 일하는 느낌을 만드는 기반입니다.

## 2. 장기기억

오래 유지되는 선호와 환경 정보를 저장합니다. 단, 비밀값과 일회성 작업 기록은 저장하지 않습니다.

## 3. 스킬

반복 가능한 절차와 장애 대응법을 저장합니다. 실패를 재발 방지 체크리스트로 바꾸는 계층입니다.

## 4. 도메인 아카이브

큰 자료 묶음은 별도 폴더와 문서로 관리합니다. AI는 필요할 때 검색하고 읽어서 답변에 반영합니다.

## 5. 자동화 루프

정기 보고, 자료 업로드, 관심사 모니터링 같은 반복 작업은 cron과 script로 운영합니다.

## 6. 주간 회고

매주 성장 로그를 작성하면 개인 AI 비서의 변화가 추적 가능해지고, 재설치나 복제도 쉬워집니다.
