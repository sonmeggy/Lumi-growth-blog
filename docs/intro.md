# 루미 성장 기록

이 사이트는 Hermes Agent 기반 개인 AI 비서 **루미**를 어떻게 성장시키는지 정리하는 공개 아카이브입니다.

핵심 메시지는 단순합니다.

> AI를 직접 재학습시키지 않아도, 페르소나·장기기억·스킬·도메인 아카이브·자동화 루프를 설계하면 개인 AI 비서는 일주일 단위로 성장한다.

## 이 사이트에서 다루는 것

- 개인 AI 비서를 처음 설정하는 방법
- 기억과 스킬을 나누는 기준
- 반복 업무를 자동화하는 방법
- 실패를 다음 성장으로 바꾸는 운영 방식
- 공개 가능한 성장 로그를 만드는 방법

## 공개 원칙

이 사이트는 운영 방법론만 공개합니다. 개인 계정, 토큰, 사적인 주소, 회사 내부 정보, 투자 세부 내역 같은 민감정보는 포함하지 않습니다.
