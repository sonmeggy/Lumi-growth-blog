# 개인 AI 비서 부트스트랩 체크리스트

## 1. 기본 설치와 상태 확인

- Hermes Agent 설치
- 모델/provider 설정
- 메시징 gateway 연결
- 필요한 toolset 활성화
- 작은 도구 사용 작업으로 end-to-end 테스트

## 2. 페르소나 정의

- 이름과 호칭
- 개인 요청과 전문 요청의 말투 차이
- 답변 형식
- 위험 작업 전 확인 규칙
- 외부 콘텐츠와 비밀값 처리 원칙

## 3. 장기기억 원칙 세우기

저장할 것:

- 오래 유지되는 사용자 선호
- 반복적으로 필요한 환경 정보
- 도메인별 운영 convention

저장하지 않을 것:

- API 키와 토큰
- 일회성 작업 결과
- 곧 바뀔 진행상황
- 민감한 개인정보

## 4. 반복 절차는 스킬로 만들기

반복되는 업무, 장애 대응, 문서 작성 형식은 스킬로 승격합니다. 한 번 해결한 문제를 다음에도 빨리 해결하기 위한 절차 저장소입니다.

## 5. 도메인 아카이브 만들기

큰 자료는 장기기억에 넣지 않고 별도 아카이브로 둡니다.

예시:

- 학습자료
- 투자 리포트 구조
- 회의록과 업무 메모
- 자동화 스크립트

## 6. 매주 회고하기

- 새로 가능해진 능력
- 새로 만든 스킬
- 자동화 변화
- 실패와 교훈
- 다음 주 개선 후보
